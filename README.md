# 🐍 Python for Data

This repository is my personal workspace for learning and practicing **Python** in the context of **Data Analysis, Data Science, and Data Engineering**.

It contains code snippets, small projects, and exercises I'm working through as I transition from Data Analyst to Data Engineer.

---

## 📁 What's Inside

| Folder | Focus Area | Examples |
|--------|------------|----------|
| `data_analysis/` | Data cleaning, exploration, and visualization | Pandas, Matplotlib, Seaborn |
| `data_science/` | Basic modeling, statistics, ML fundamentals | Scikit-learn, regression, classification |
| `data_engineering/` | Pipelines, ETL, data structures | Python scripting, file handling, APIs |
| `practice/` | Small exercises and katas | Loops, functions, list comprehensions |

> *Note: Folders will be filled as I learn. This is a living repo.*
